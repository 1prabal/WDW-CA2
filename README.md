# Student Portfolio

A multi-page personal portfolio showcasing home, about, skills, and contact sections.

## Pages

- **Home** (`index.html`) — landing page introducing the portfolio.
- **About** (`about.html`) — personal introduction with profile image.
- **Skills** (`skills.html`) — highlights skills and mini projects using cards.
- **Contact** (`contact.html`) — contact form for inquiries.

## Styles

Global and page-specific styling is managed in `style.css`, covering layout, navigation bar, cards, and forms.

## Usage

Open `index.html` in a browser to navigate the site via the top navbar.
